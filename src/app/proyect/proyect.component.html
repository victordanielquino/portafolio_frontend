  • Proyecto: Pizarra Digital Web

    Esta es una aplicación web destina a ayudar a estudiantes de colegios y profesores del area de matemáticas. Las vondades de esta aplicacion se resumen en juntar algunas características de la libreria MathIO de Google para generar texto Matemático, generacion del plano cartesiano y graficos de geomentria analítica de manera dinamica también cuenta con un canvas donde se puede rayar. Esta aplicación esta desarrollada bajo la libreria de React haciendo uso de Hooks, se usa Sass y Material UI para el manejo de estilo y componentes, tambien se uso Firebase de Google para almacenar imagenes y pdfs.
