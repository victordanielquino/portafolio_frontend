Title:"PIZARRA DIGITAL WEB"

Aplication: link

Repository:GitHub

Description:

Esta es una aplicación web destina a ayudar a estudiantes de colegios y profesores del area de matemáticas. Las vondades de esta aplicacion se resumen en juntar algunas características de google y la api mathio, la generacion de grificos de geométria análitica de geogebra y la pizarra de OpenBoard. Esta aplicación esta desarrollada bajo la libreria de React, se usa Sass y Material UI para el manejo de estilo y componentes, se uso Firebase de Google para almacenar imagenes y pdfs. La aplicación puede generar un reporte PDF desde el front.
